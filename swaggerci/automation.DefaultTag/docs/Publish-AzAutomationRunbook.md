# Publish-AzAutomationRunbook

## SYNOPSIS
Publish runbook draft.

## SYNTAX

### Publish (Default)
```
Publish-AzAutomationRunbook -AutomationAccountName <String> -Name <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### PublishViaIdentity
```
Publish-AzAutomationRunbook -InputObject <IAutomationIdentity> [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Publish runbook draft.
