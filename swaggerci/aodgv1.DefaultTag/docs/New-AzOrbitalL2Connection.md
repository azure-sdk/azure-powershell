# New-AzOrbitalL2Connection

## SYNOPSIS
Creates or updates an L2 Connection.

## SYNTAX

```
New-AzOrbitalL2Connection -Name <String> -ResourceGroupName <String> -EdgeSiteId <String>
 -EdgeSitePartnerRouterName <String> -GroundStationId <String> -GroundStationPartnerRouterName <String>
 -Location <String> -VlanId <Int32> [-SubscriptionId <String>] [-Tag <Hashtable>] [-DefaultProfile <PSObject>]
 [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
