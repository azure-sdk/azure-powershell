# Get-AzStorSimpleManagerEncryptionKey

## SYNOPSIS
Returns the symmetric encryption key of the manager.

## SYNTAX

### Get (Default)
```
Get-AzStorSimpleManagerEncryptionKey -ManagerName <String> -ResourceGroupName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### GetViaIdentity
```
Get-AzStorSimpleManagerEncryptionKey -InputObject <IStorSimpleIdentity> [-DefaultProfile <PSObject>]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Returns the symmetric encryption key of the manager.
