# Update-AzBatchPrivateEndpointConnection

## SYNOPSIS
Updates the properties of an existing private endpoint connection.

## SYNTAX

### UpdateExpanded (Default)
```
Update-AzBatchPrivateEndpointConnection -AccountName <String> -Name <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-IfMatch <String>] [-PrivateLinkServiceConnectionStateDescription <String>]
 [-PrivateLinkServiceConnectionStateStatus <PrivateLinkServiceConnectionStatus>] [-Tag <Hashtable>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### UpdateViaIdentityExpanded
```
Update-AzBatchPrivateEndpointConnection -InputObject <IBatchIdentity> [-IfMatch <String>]
 [-PrivateLinkServiceConnectionStateDescription <String>]
 [-PrivateLinkServiceConnectionStateStatus <PrivateLinkServiceConnectionStatus>] [-Tag <Hashtable>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```
