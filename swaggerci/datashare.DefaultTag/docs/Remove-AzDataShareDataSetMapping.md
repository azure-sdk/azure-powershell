# Remove-AzDataShareDataSetMapping

## SYNOPSIS
Delete a DataSetMapping in a shareSubscription

## SYNTAX

### Delete (Default)
```
Remove-AzDataShareDataSetMapping -AccountName <String> -Name <String> -ResourceGroupName <String>
 -ShareSubscriptionName <String> [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-PassThru]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzDataShareDataSetMapping -InputObject <IDataShareIdentity> [-DefaultProfile <PSObject>] [-PassThru]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Delete a DataSetMapping in a shareSubscription
