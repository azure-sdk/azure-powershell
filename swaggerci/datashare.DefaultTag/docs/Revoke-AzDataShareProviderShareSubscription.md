# Revoke-AzDataShareProviderShareSubscription

## SYNOPSIS
Revoke share subscription in a provider share

## SYNTAX

### Revoke (Default)
```
Revoke-AzDataShareProviderShareSubscription -AccountName <String> -Id <String> -ResourceGroupName <String>
 -ShareName <String> [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

### RevokeViaIdentity
```
Revoke-AzDataShareProviderShareSubscription -InputObject <IDataShareIdentity> [-DefaultProfile <PSObject>]
 [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```
