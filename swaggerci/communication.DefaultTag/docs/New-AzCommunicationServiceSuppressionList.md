# New-AzCommunicationServiceSuppressionList

## SYNOPSIS
Add a new SuppressionList resource under the parent Domains resource or update an existing SuppressionList resource.

## SYNTAX

```
New-AzCommunicationServiceSuppressionList -DomainName <String> -EmailServiceName <String> -Name <String>
 -ResourceGroupName <String> [-SubscriptionId <String>] [-ListName <String>] [-DefaultProfile <PSObject>]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Add a new SuppressionList resource under the parent Domains resource or update an existing SuppressionList resource.
