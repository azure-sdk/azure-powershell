# Invoke-AzCostOrderGenerateBenefitUtilizationSummaryReportAsync

## SYNOPSIS
Triggers generation of a benefit utilization summaries report for the provided reservation order.

## SYNTAX

### Order (Default)
```
Invoke-AzCostOrderGenerateBenefitUtilizationSummaryReportAsync -ReservationOrderId <String> [-Filter <String>]
 [-Grain <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### OrderViaIdentity
```
Invoke-AzCostOrderGenerateBenefitUtilizationSummaryReportAsync -InputObject <ICostIdentity> [-Filter <String>]
 [-Grain <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Triggers generation of a benefit utilization summaries report for the provided reservation order.
