# Update-AzDashboardManagedPrivateEndpoint

## SYNOPSIS
Refresh and sync managed private endpoints of a grafana resource to latest state.

## SYNTAX

### Refresh (Default)
```
Update-AzDashboardManagedPrivateEndpoint -ResourceGroupName <String> -WorkspaceName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### RefreshViaIdentity
```
Update-AzDashboardManagedPrivateEndpoint -InputObject <IDashboardIdentity> [-DefaultProfile <PSObject>]
 [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### UpdateExpanded
```
Update-AzDashboardManagedPrivateEndpoint -Name <String> -ResourceGroupName <String> -WorkspaceName <String>
 [-SubscriptionId <String>] [-Tag <Hashtable>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

### UpdateViaIdentityExpanded
```
Update-AzDashboardManagedPrivateEndpoint -InputObject <IDashboardIdentity> [-Tag <Hashtable>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Refresh and sync managed private endpoints of a grafana resource to latest state.
