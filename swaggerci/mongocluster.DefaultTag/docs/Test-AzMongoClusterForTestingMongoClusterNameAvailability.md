# Test-AzMongoClusterForTestingMongoClusterNameAvailability

## SYNOPSIS
Check if mongo cluster name is available for use.

## SYNTAX

### CheckExpanded (Default)
```
Test-AzMongoClusterForTestingMongoClusterNameAvailability -Location <String> [-SubscriptionId <String>]
 [-Name <String>] [-Type <String>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### Check
```
Test-AzMongoClusterForTestingMongoClusterNameAvailability -Location <String>
 -Body <ICheckNameAvailabilityRequest> [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```
