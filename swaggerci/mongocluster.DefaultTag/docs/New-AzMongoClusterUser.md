# New-AzMongoClusterUser

## SYNOPSIS
Creates a new user or updates an existing user on a mongo cluster.

## SYNTAX

```
New-AzMongoClusterUser -MongoClusterName <String> -Name <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-Role <IDatabaseRole[]>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Creates a new user or updates an existing user on a mongo cluster.
