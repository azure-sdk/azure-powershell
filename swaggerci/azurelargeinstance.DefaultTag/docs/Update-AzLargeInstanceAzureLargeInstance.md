# Update-AzLargeInstanceAzureLargeInstance

## SYNOPSIS
Patches the Tags field of an Azure Large Instance for the specified subscription, resource group, and instance name.

## SYNTAX

### UpdateExpanded (Default)

```
Update-AzLargeInstanceAzureLargeInstance -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-Tag <Hashtable>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```
