# Start-AzContainerServiceFleetUpdateRun

## SYNOPSIS
Starts an UpdateRun.

## SYNTAX

### Start (Default)
```
Start-AzContainerServiceFleetUpdateRun -FleetName <String> -Name <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-IfMatch <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

### StartViaIdentity
```
Start-AzContainerServiceFleetUpdateRun -InputObject <IContainerServiceFleetIdentity> [-IfMatch <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```
