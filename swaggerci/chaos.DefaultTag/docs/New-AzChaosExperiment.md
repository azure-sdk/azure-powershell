# New-AzChaosExperiment

## SYNOPSIS
Create or update a Experiment resource.

## SYNTAX

```
New-AzChaosExperiment -Name <String> -ResourceGroupName <String> -Location <String>
 -Selector <IChaosTargetSelector[]> -Step <IChaosExperimentStep[]> [-SubscriptionId <String>]
 [-CustomerDataStorageAccountResourceId <String>] [-CustomerDataStorageBlobContainerName <String>]
 [-IdentityType <ManagedServiceIdentityType>] [-IdentityUserAssignedIdentity <Hashtable>] [-Tag <Hashtable>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```
