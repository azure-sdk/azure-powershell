# Disable-AzMarketplaceRpServicePrivateStoreCollectionApproveItem

## SYNOPSIS

Disable-AzMarketplaceRpServicePrivateStoreCollectionApproveItem -CollectionId <String>
 -PrivateStoreId <String> [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

```
Disable-AzMarketplaceRpServicePrivateStoreCollectionApproveItem -InputObject <IMarketplaceRpServiceIdentity>
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
