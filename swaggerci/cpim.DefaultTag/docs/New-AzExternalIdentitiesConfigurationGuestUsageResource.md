# New-AzExternalIdentitiesConfigurationGuestUsageResource

## SYNOPSIS
Creates a Guest Usages resource, which is used to linking a subscription to an instance of Azure AD External Identities.
[Learn more](https://aka.ms/extidbilling).

## SYNTAX

```
New-AzExternalIdentitiesConfigurationGuestUsageResource -ResourceGroupName <String> -ResourceName <String>
 -Location <String> [-SubscriptionId <String>] [-Tag <Hashtable>] [-TenantId <String>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Creates a Guest Usages resource, which is used to linking a subscription to an instance of Azure AD External Identities.
[Learn more](https://aka.ms/extidbilling).
