# New-AzAppPlatformJob

## SYNOPSIS
Create a new Job or update an exiting Job.

## SYNTAX

```
New-AzAppPlatformJob -Name <String> -ResourceGroupName <String> -ServiceName <String>
 [-SubscriptionId <String>] [-AddonConfig <Hashtable>] [-EnvironmentVariableProperty <Hashtable>]
 [-EnvironmentVariableSecret <Hashtable>] [-SourceType <String>] [-SourceVersion <String>]
 [-TemplateArg <String[]>] [-TemplateParallelism <Int32>] [-TemplateReplicaCompletionCount <Int32>]
 [-TemplateReplicaRetryLimit <Int32>] [-TemplateReplicaTimeout <Int32>] [-TriggerConfigTriggerType <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
