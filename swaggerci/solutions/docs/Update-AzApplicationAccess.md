# Update-AzApplicationAccess

## SYNOPSIS
Update access for application.

## SYNTAX

### UpdateExpanded (Default)
```
Update-AzApplicationAccess -ApplicationName <String> -ResourceGroupName <String> -Status <Status>
 -SubStatus <Substatus> [-SubscriptionId <String>] [-Approver <String>] [-MetadataOriginRequestId <String>]
 [-MetadataRequestorId <String>] [-MetadataSubjectDisplayName <String>] [-MetadataTenantDisplayName <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

```
 [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```
