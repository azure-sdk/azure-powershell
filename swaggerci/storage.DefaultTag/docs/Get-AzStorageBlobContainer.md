# Get-AzStorageBlobContainer

## SYNOPSIS
Gets properties of a specified container.

## SYNTAX

### List (Default)
```
Get-AzStorageBlobContainer -AccountName <String> -ResourceGroupName <String> [-SubscriptionId <String[]>]
 [-Filter <String>] [-Include <ListContainersInclude>] [-Maxpagesize <String>] [-DefaultProfile <PSObject>]
 [<CommonParameters>]
```

### Get
```
Get-AzStorageBlobContainer -AccountName <String> -ContainerName <String> -ResourceGroupName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### GetViaIdentity
```
Get-AzStorageBlobContainer -InputObject <IStorageIdentity> [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## DESCRIPTION
Gets properties of a specified container.
