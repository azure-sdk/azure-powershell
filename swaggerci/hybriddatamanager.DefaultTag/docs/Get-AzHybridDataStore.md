# Get-AzHybridDataStore

## SYNOPSIS
This method gets the data store/repository by name.

## SYNTAX

### List (Default)
```
Get-AzHybridDataStore -DataManagerName <String> -ResourceGroupName <String> [-SubscriptionId <String[]>]
 [-Filter <String>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### Get
```
Get-AzHybridDataStore -DataManagerName <String> -Name <String> -ResourceGroupName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### GetViaIdentity
```
Get-AzHybridDataStore -InputObject <IHybridIdentity> [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## DESCRIPTION
This method gets the data store/repository by name.
