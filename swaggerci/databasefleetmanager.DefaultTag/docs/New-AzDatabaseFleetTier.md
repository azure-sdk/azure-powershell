# New-AzDatabaseFleetTier

## SYNOPSIS
Creates or updates a tier.

## SYNTAX

```
New-AzDatabaseFleetTier -FleetName <String> -ResourceGroupName <String> -TierName <String>
 [-SubscriptionId <String>] [-Capacity <Int32>] [-DatabaseCapacityMax <Double>]
 [-DatabaseCapacityMin <Double>] [-DatabaseSizeGbMax <Int32>] [-Family <String>]
 [-HighAvailabilityReplicaCount <Int32>] [-Pooled] [-PoolNumOfDatabasesMax <Int32>] [-Serverless]
 [-ServiceTier <String>] [-ZoneRedundancy <ZoneRedundancy>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```
