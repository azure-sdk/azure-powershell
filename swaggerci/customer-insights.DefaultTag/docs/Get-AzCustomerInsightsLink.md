# Get-AzCustomerInsightsLink

## SYNOPSIS
Gets a link in the hub.

## SYNTAX

### List (Default)
```
Get-AzCustomerInsightsLink -HubName <String> -ResourceGroupName <String> [-SubscriptionId <String[]>]
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

```
Get-AzCustomerInsightsLink -HubName <String> -Name <String> -ResourceGroupName <String>

```
