# Remove-AzDevTestLabsArtifactSource

## SYNOPSIS

## SYNTAX

### Delete (Default)
```
Remove-AzDevTestLabsArtifactSource -LabName <String> -Name <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzDevTestLabsArtifactSource -InputObject <IDevTestLabsIdentity> [-DefaultProfile <PSObject>]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```
