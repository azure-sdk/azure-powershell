# New-AzNetworkAnalyticsDataConnector

## SYNOPSIS

## SYNTAX

```
New-AzNetworkAnalyticsDataConnector -Name <String> -ResourceGroupName <String> -Location <String>
 [-SubscriptionId <String>] [-BronzeStorageAccountRetentionDay <Int32>] [-ComputeFailureAlertEmail <String>]
 [-ComputeThroughput <Int32>] [-DataExplorerRetentionDay <Int32>] [-DataLakeHouse <String[]>]
 [-DatasetColumnsToMask <String[]>] [-DatasetName <String>] [-DatasetTimestampKey <DateTime>]
 [-GoldStorageAccountRetentionDay <Int32>] [-ManagedResourceGroupConfigurationLocation <String>]
 [-ManagedResourceGroupConfigurationName <String>] [-ProductConnector <String>]
 [-ProductConnectorDataset <String>] [-ProductName <String>] [-ProductVersion <String>]
 [-SilverStorageAccountRetentionDay <Int32>] [-Tag <Hashtable>] [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```
