# New-AzCognitiveServicesRaiBlocklist

## SYNOPSIS
Update the state of specified blocklist associated with the Azure OpenAI account.

## SYNTAX

```
New-AzCognitiveServicesRaiBlocklist -AccountName <String> -Name <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-Description <String>] [-Tag <Hashtable>] [-DefaultProfile <PSObject>] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Update the state of specified blocklist associated with the Azure OpenAI account.
