# New-AzProfessionalServiceForEnrollment

## SYNOPSIS
Creates or updates a ProfessionalService resource for enrollment account.

## SYNTAX

```
New-AzProfessionalServiceForEnrollment -EnrollmentBillingAccount <String> -ResourceName <String> [-AutoRenew]
 [-Location <String>] [-Name <String>] [-OfferId <String>] [-PaymentChannelMetadata <Hashtable>]
 [-PublisherId <String>] [-Quantity <Single>] [-QuoteId <String>] [-SkuId <String>] [-StoreFront <String>]
 [-Tag <Hashtable>] [-TermId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```
