# Remove-AzProfessionalService

## SYNOPSIS
Deletes the specified ProfessionalService.

## SYNTAX

### Delete (Default)
```
Remove-AzProfessionalService -BillingAccount <String> -BillingProfile <String> -InvoiceSection <String>
 -ResourceName <String> [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzProfessionalService -InputObject <IProfessionalServiceIdentity> [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Deletes the specified ProfessionalService.
