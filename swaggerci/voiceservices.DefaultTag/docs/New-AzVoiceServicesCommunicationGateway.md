# New-AzVoiceServicesCommunicationGateway

## SYNOPSIS
Create a CommunicationsGateway

## SYNTAX

```
New-AzVoiceServicesCommunicationGateway -CommunicationsGatewayName <String> -ResourceGroupName <String>
 -Location <String> [-SubscriptionId <String>] [-ApiBridgeAllowedAddressPrefix <String[]>]
 [-ApiBridgeConfigureApiBridge <ApiBridgeActivationState>]
 [-AutoGeneratedDomainNameLabelScope <AutoGeneratedDomainNameLabelScope>] [-Codec <TeamsCodecs[]>]
 [-CustomSipHeader <ICustomSipHeader[]>] [-DnsDelegation <IDnsDelegationProperties[]>] [-E911Type <E911Type>]
 [-EmergencyDialString <String[]>] [-IdentityType <ManagedServiceIdentityType>]
 [-IdentityUserAssignedIdentity <Hashtable>] [-IntegratedMcpEnabled] [-OnPremMcpEnabled]
 [-Platform <CommunicationsPlatform[]>] [-ServiceLocation <IServiceRegionProperties[]>] [-SkuCapacity <Int32>]
 [-SkuFamily <String>] [-SkuName <String>] [-SkuSize <String>] [-SkuTier <SkuTier>] [-Tag <Hashtable>]
 [-TeamsVoicemailPilotNumber <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

## DESCRIPTION
Create a CommunicationsGateway
