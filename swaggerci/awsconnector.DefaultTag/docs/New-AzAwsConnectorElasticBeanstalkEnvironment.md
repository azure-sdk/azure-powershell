# New-AzAwsConnectorElasticBeanstalkEnvironment

## SYNOPSIS
Create a ElasticBeanstalkEnvironment.
Please note that end user resource creation is currently restricted, but it will be enabled in the future.

## SYNTAX

```
New-AzAwsConnectorElasticBeanstalkEnvironment -Name <String> -ResourceGroupName <String> -Location <String>
 [-SubscriptionId <String>] [-Arn <String>] [-AwPropertyApplicationName <String>]
 [-AwPropertyCnamePrefix <String>] [-AwPropertyDescription <String>] [-AwPropertyEndpointUrl <String>]
 [-AwPropertyEnvironmentName <String>] [-AwPropertyOperationsRole <String>]
 [-AwPropertyOptionSetting <IOptionSetting[]>] [-AwPropertyPlatformArn <String>]
 [-AwPropertySolutionStackName <String>] [-AwPropertyTag <ITagAutoGenerated29[]>]
 [-AwPropertyTemplateName <String>] [-AwPropertyVersionLabel <String>] [-AwsAccountId <String>]
 [-AwsRegion <String>] [-AwsSourceSchema <String>] [-AwsTag <Hashtable>]
 [-PublicCloudConnectorsResourceId <String>] [-PublicCloudResourceName <String>] [-Tag <Hashtable>]
 [-TierName <String>] [-TierType <String>] [-TierVersion <String>] [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Create a ElasticBeanstalkEnvironment.
Please note that end user resource creation is currently restricted, but it will be enabled in the future.
