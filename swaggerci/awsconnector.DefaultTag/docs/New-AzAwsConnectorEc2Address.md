# New-AzAwsConnectorEc2Address

## SYNOPSIS
Create a Ec2Address.
Please note that end user resource creation is currently restricted, but it will be enabled in the future.

## SYNTAX

```
New-AzAwsConnectorEc2Address -Name <String> -ResourceGroupName <String> -Location <String>
 [-SubscriptionId <String>] [-Arn <String>] [-AwPropertyAllocationId <String>]
 [-AwPropertyAssociationId <String>] [-AwPropertyCarrierIP <String>] [-AwPropertyCustomerOwnedIP <String>]
 [-AwPropertyCustomerOwnedIpv4Pool <String>] [-AwPropertyInstanceId <String>]
 [-AwPropertyNetworkBorderGroup <String>] [-AwPropertyNetworkInterfaceId <String>]
 [-AwPropertyNetworkInterfaceOwnerId <String>] [-AwPropertyPrivateIPAddress <String>]
 [-AwPropertyPublicIP <String>] [-AwPropertyPublicIpv4Pool <String>] [-AwPropertyTag <ITagAutoGenerated10[]>]
 [-AwsAccountId <String>] [-AwsRegion <String>] [-AwsSourceSchema <String>] [-AwsTag <Hashtable>]
 [-DomainValue <DomainType>] [-PublicCloudConnectorsResourceId <String>] [-PublicCloudResourceName <String>]
 [-Tag <Hashtable>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Create a Ec2Address.
Please note that end user resource creation is currently restricted, but it will be enabled in the future.
