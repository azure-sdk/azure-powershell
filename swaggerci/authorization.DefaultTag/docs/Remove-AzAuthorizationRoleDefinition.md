# Remove-AzAuthorizationRoleDefinition

## SYNOPSIS
Deletes a role definition.

## SYNTAX

### Delete (Default)
```
Remove-AzAuthorizationRoleDefinition -Id <String> -Scope <String> [-DefaultProfile <PSObject>] [-PassThru]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzAuthorizationRoleDefinition -InputObject <IAuthorizationIdentity> [-DefaultProfile <PSObject>]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Deletes a role definition.
