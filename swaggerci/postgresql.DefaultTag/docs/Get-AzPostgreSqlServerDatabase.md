# Get-AzPostgreSqlServerDatabase

## SYNOPSIS
Gets information about a database.

## SYNTAX

### List (Default)
```
Get-AzPostgreSqlServerDatabase -ResourceGroupName <String> -ServerName <String> [-SubscriptionId <String[]>]
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### Get
```
Get-AzPostgreSqlServerDatabase -Name <String> -ResourceGroupName <String> -ServerName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### GetViaIdentity
```
Get-AzPostgreSqlServerDatabase -InputObject <IPostgreSqlServerIdentity> [-DefaultProfile <PSObject>]
 [<CommonParameters>]
```
