# New-AzPostgreSqlVirtualEndpoint

## SYNOPSIS
Creates a new virtual endpoint for PostgreSQL flexible server.

## SYNTAX

```
New-AzPostgreSqlVirtualEndpoint -Name <String> -ResourceGroupName <String> -ServerName <String>
 [-SubscriptionId <String>] [-EndpointType <VirtualEndpointType>] [-Member <String[]>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Creates a new virtual endpoint for PostgreSQL flexible server.
