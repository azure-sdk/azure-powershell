# Get-AzServiceLinkerConnector

## SYNOPSIS
Returns Connector resource for a given name.

## SYNTAX

### List (Default)
```
Get-AzServiceLinkerConnector -Location <String> -ResourceGroupName <String> [-SubscriptionId <String[]>]
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### Get
```
Get-AzServiceLinkerConnector -Location <String> -Name <String> -ResourceGroupName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### GetViaIdentity
```
Get-AzServiceLinkerConnector -InputObject <IServiceLinkerIdentity> [-DefaultProfile <PSObject>]
 [<CommonParameters>]
```
