# Update-AzServiceLinkerDryrun

## SYNOPSIS

## SYNTAX

### UpdateExpanded (Default)
```
Update-AzServiceLinkerDryrun -DryrunName <String> -ResourceUri <String> [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### UpdateViaIdentityExpanded
