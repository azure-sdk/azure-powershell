# New-AzApiCenterDeployment

## SYNOPSIS
Creates new or updates existing API deployment.

## SYNTAX

```
New-AzApiCenterDeployment -ApiName <String> -Name <String> -ResourceGroupName <String> -ServiceName <String>
 -WorkspaceName <String> [-SubscriptionId <String>] [-CustomProperty <IAny>] [-DefinitionId <String>]
 [-Description <String>] [-EnvironmentId <String>] [-ServerRuntimeUri <String[]>] [-State <DeploymentState>]
 [-Title <String>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
