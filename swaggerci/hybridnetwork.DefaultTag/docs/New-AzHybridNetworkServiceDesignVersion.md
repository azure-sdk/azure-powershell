# New-AzHybridNetworkServiceDesignVersion

## SYNOPSIS
Creates or updates a network service design version.

 -PublisherName <String> -ResourceGroupName <String> -Location <String> [-SubscriptionId <String>]
 [-ConfigurationGroupSchemaReference <Hashtable>] [-Description <String>] [-NfvisFromSite <Hashtable>]
 [-ResourceElementTemplate <IResourceElementTemplate[]>] [-Tag <Hashtable>] [-DefaultProfile <PSObject>]
 [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
