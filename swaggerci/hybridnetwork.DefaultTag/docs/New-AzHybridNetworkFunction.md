# New-AzHybridNetworkFunction

## SYNOPSIS
Creates or updates a network function resource.

## SYNTAX

```
New-AzHybridNetworkFunction -Name <String> -ResourceGroupName <String> -Location <String>
 [-SubscriptionId <String>] [-AllowSoftwareUpdate] [-ConfigurationType <NetworkFunctionConfigurationType>]
 [-Etag <String>] [-IdentityType <ManagedServiceIdentityType>] [-IdentityUserAssignedIdentity <Hashtable>]
 [-NetworkFunctionDefinitionGroupName <String>] [-NetworkFunctionDefinitionOfferingLocation <String>]
 [-NetworkFunctionDefinitionVersion <String>]
 [-NetworkFunctionDefinitionVersionResourceReferenceIdType <IdType>] [-NfviId <String>] [-NfviType <NfviType>]
 [-PublisherName <String>] [-PublisherScope <PublisherScope>] [-RoleOverrideValue <String[]>]
 [-Tag <Hashtable>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```
